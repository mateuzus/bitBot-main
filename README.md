### 🎯 Objetivo
Monitorar o valor da moeda para ajudar possíveis Daytraders no dia a dia.

### 🔎 Como utilizar

- Clonar o rep.
- Executar o comando abaixo.
- `npm install`
- Executar o arquivo `bitSearch.js` com `node bitSearch.js`
